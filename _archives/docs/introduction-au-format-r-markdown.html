12.2 Introduction à GitHub

Le system de suivi de version git a été créé par Linus Torvalds, aussi connu pour avoir créé Linux. Il existe plusieurs véhicules pour rendre git disponible en ligne, mais GitHub est définitivement le plus utilisé d’entre tous. La plateforme GitHub, maintenant la propriété de Microsoft, n’est en soi pas open source. Si cela vous pose problème, je vous redirige vers GitLab, qui fonctionne à peu près de la même manière que GitHub (alors que la plateforme GitHub sera fort probablement toujours vivante dans plusieurs années, on en est moins sûr pour GitLab, c’est pourquoi j’archive plutôt mes calculs sur GitHub).
